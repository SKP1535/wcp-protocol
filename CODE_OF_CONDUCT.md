# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in the WCP project a respectful and inclusive experience for everyone.

---

## Our Standards

Examples of positive behavior include:

* being respectful and constructive
* welcoming new contributors
* focusing on collaboration
* providing helpful feedback

Examples of unacceptable behavior include:

* harassment or discrimination
* personal attacks
* trolling or disruptive behavior
* publishing private information without consent

---

## Enforcement

Project maintainers are responsible for enforcing this code of conduct.

They may remove or reject contributions that violate these standards.

---

## Scope

This code of conduct applies to:

* GitHub issues
* pull requests
* community discussions
* project communication channels

---

## Reporting

If you experience or witness unacceptable behavior, please report it by opening an issue or contacting the maintainers.

All reports will be reviewed and handled appropriately.

---

This Code of Conduct is inspired by common open source community standards such as the Contributor Covenant.
